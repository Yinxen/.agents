## 配置語言

在網站根目錄的 `_config.yml` 中設定：

### 單一語言

```yaml
language: zh-tw
```

### 多語言（優先順序）

```yaml
language:
  - zh-tw
  - en
  - ja
```

## 語言檔案

### 檔案位置

語言檔案存放在主題的 `languages/` 目錄：

```text
themes/your-theme/
└── languages/
    ├── default.yml   # 預設語言（選用）
    ├── en.yml        # 英文
    ├── zh-tw.yml     # 繁體中文
    └── ja.yml        # 日文
```

### 檔案格式

支援 YAML 或 JSON 格式：

```yaml
# languages/zh-tw.yml
title: 標題
subtitle: 副標題

# 巢狀結構
menu:
  home: 首頁
  archives: 歸檔
  tags: 標籤
  categories: 分類
  about: 關於

# 日期格式
date_format: YYYY年MM月DD日
time_format: HH:mm:ss

# 文章相關
post:
  created: 發表於
  updated: 更新於
  comments: 則留言
  read_more: 閱讀更多

# 分頁
pagination:
  prev: 上一頁
  next: 下一頁

# 其他
powered_by: 由 %s 驅動
```

### printf 格式

使用 `%s`、`%d` 等佔位符：

```yaml
powered_by: 由 %s 驅動
article_count: 共 %d 篇文章
```

## 模板中使用

### 基本翻譯 `__()`

```ejs
<%= __('title') %>
<!-- 輸出：標題 -->

<%= __('menu.home') %>
<!-- 輸出：首頁 -->

<%= __('powered_by', 'Hexo') %>
<!-- 輸出：由 Hexo 驅動 -->
```

### 複數形式 `_p()`

```yaml
# languages/en.yml
post:
  zero: No posts
  one: One post
  other: "%d posts"
```

```ejs
<%= _p('post', 0) %>
<!-- 輸出：No posts -->

<%= _p('post', 1) %>
<!-- 輸出：One post -->

<%= _p('post', 5) %>
<!-- 輸出：5 posts -->
```

## 路徑語言偵測

### 設定 i18n 目錄

在 `_config.yml` 中設定：

```yaml
i18n_dir: :lang
```

### 路徑對應規則

| 路徑                | 對應語言      |
| ------------------- | ------------- |
| `/index.html`       | 預設語言 (en) |
| `/zh-tw/index.html` | zh-tw         |
| `/ja/index.html`    | ja            |

### 注意事項

- 僅在對應語言檔存在時才生效
- 語言代碼必須與檔案名稱完全匹配

## 實用範例

### 導覽選單

```yaml
# languages/zh-tw.yml
menu:
  home: 首頁
  archives: 文章歸檔
  categories: 分類
  tags: 標籤
  about: 關於我
```

```ejs
<nav>
  <% for (var name in theme.menu) { %>
    <a href="<%- url_for(theme.menu[name]) %>">
      <%= __('menu.' + name) %>
    </a>
  <% } %>
</nav>
```

### 文章資訊

```yaml
# languages/zh-tw.yml
post:
  posted_on: 發表於
  updated_on: 更新於
  filed_under: 分類於
  tagged_with: 標籤
```

```ejs
<div class="post-meta">
  <span><%= __('post.posted_on') %> <%= date(page.date, 'YYYY-MM-DD') %></span>
  <% if (page.updated) { %>
    <span><%= __('post.updated_on') %> <%= date(page.updated, 'YYYY-MM-DD') %></span>
  <% } %>
  <% if (page.categories && page.categories.length) { %>
    <span>
      <%= __('post.filed_under') %>
      <% page.categories.each(function(cat, i){ %>
        <a href="<%- url_for(cat.path) %>"><%= cat.name %></a><%= i < page.categories.length - 1 ? ', ' : '' %>
      <% }) %>
    </span>
  <% } %>
</div>
```

### 分頁導覽

```yaml
# languages/zh-tw.yml
pagination:
  prev: ← 較新文章
  next: 較舊文章 →
```

```ejs
<%- paginator({
  prev_text: __('pagination.prev'),
  next_text: __('pagination.next')
}) %>
```

### 搜尋表單

```yaml
# languages/zh-tw.yml
search:
  placeholder: 搜尋...
  button: 搜尋
```

```ejs
<form class="search-form">
  <input type="text" placeholder="<%= __('search.placeholder') %>">
  <button type="submit"><%= __('search.button') %></button>
</form>
```

## 多語言網站結構

### 目錄結構建議

```text
source/
├── _posts/           # 預設語言文章
├── zh-tw/
│   └── _posts/       # 繁體中文文章
└── ja/
    └── _posts/       # 日文文章
```

### 語言切換器

```ejs
<div class="language-switcher">
  <a href="<%- url_for('/') %>" class="<%= config.language === 'en' ? 'active' : '' %>">EN</a>
  <a href="<%- url_for('/zh-tw/') %>" class="<%= config.language === 'zh-tw' ? 'active' : '' %>">中文</a>
  <a href="<%- url_for('/ja/') %>" class="<%= config.language === 'ja' ? 'active' : '' %>">日本語</a>
</div>
```
